GraphQL Request Flow

SEQUENCE 
sequenceDiagram
    participant Test as Test File
    participant Ops as Operations Layer
    participant Mut as Auto-Generated Mutation
    participant Client as GraphQLClient
    participant Auth as Auth Fixture
    participant API as Backend /graphql

    Test->>Ops: cart_ops.add_item_to_cart(payload)
    Ops->>Ops: Build variables + attach CART_FRAGMENT
    Ops->>Mut: AddItemMutation(client).execute(vars, fields)
    Mut->>Client: client.execute(gql_query, variables)
    Client->>Auth: get_auth_headers()
    Auth-->>Client: Authorization: Bearer JWT
    Client->>API: POST /graphql
    API-->>Client: JSON response
    Client-->>Mut: Parsed data
    Mut-->>Ops: CartType result
    Ops-->>Test: dict with cart data
    Note over Test: assert cart["itemsQuantity"] == 1

E2E Test Execution Flow

SEQUENCE 
sequenceDiagram
    participant Test as E2E Test
    participant PO as Page Object
    participant Comp as Component
    participant PW as Playwright
    participant UI as Storefront UI
    participant Report as Allure Report

    Test->>PO: CartPage(page, config)
    PO->>PO: Init locators + child components
    Test->>PO: cart_page.navigate()
    PO->>PW: page.goto(url)
    PW->>UI: HTTP GET
    UI-->>PW: HTML rendered

    Test->>PO: get_line_item_by_sku("SKU-001")
    PO-->>Test: LineItemComponent

    Test->>Comp: item.quantity_stepper.increase()
    Comp->>PW: locator.click()
    PW->>UI: Click event

    Test->>Test: expect(item.quantity).to_have_text("2")

    alt Test Failure
        PW->>PW: page.screenshot(full_page=True)
        PW-->>Report: Attach screenshot
    end
